# Analiza efektywności sprzedaży - Projekt Zaliczeniowy

### Opis projektu
Repozytorium zawiera projekt zaliczeniowy dotyczący analizy danych sprzedażowych sklepu z elektroniką (listopad-grudzień 2023).

### Zawartość
W repozytorium znajdują się:
1. **Plik Excel (.xlsx)** – zawierający surowe dane, obliczenia oraz wykresy (kolumnowy i kołowy).
2. **Raport PDF** – gotowy dokument z wnioskami i sformatowanymi wykresami.
